## Javascript SDK
Here's how to send your first `identify` event. This can be sent on sign up or from an existing database of users.
``` javascript
import StarnetDataNode from '@starnet-platform/starnet-data-node'; 

secret = env.process.DATA_NODE_SECRET // provided by Starnet
sdk = new StarnetDataNode(secret)

sdk.eventRegistry.identify(profileData)
```

Interface for `profileData` above:
``` javascript
interface ProfileData {
  uniqueId: string;
  demographic: DemographicData;
  social: SocialIdentifers;
  device: DeviceData;
  interest: string[];
}
```
